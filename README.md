# Student Management System

## Project Description
This project is a Java-based Student Management System developed using IntelliJ IDEA.

## Features
- Add Student
- Show Students
- Calculate Average Grade
- Search Student
- Remove Student
- Find Top Student

## Technologies Used
- Java
- IntelliJ IDEA
- GitHub

## How to Run
1. Open project in IntelliJ IDEA
2. Run Main.java
3. Use menu options
